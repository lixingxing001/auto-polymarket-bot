# 误差分层与失败模式诊断

## 目标

把一个总准确率拆成可行动的局部地形，找出模型真正脆弱的场景。

## 当前切片维度

- 模型置信度
- 市场价格置信度
- 模型是否与市场方向一致
- 1 分钟绝对收益
- 5 分钟绝对收益
- 5 分钟实现波动率
- 5 分钟区间振幅
- 距离 barrier 的幅度

## 当前用途

- 找出“高置信但仍然错”的危险区
- 找出模型只在某些市场状态里有优势的证据
- 为下一批候选策略提供更像样的假设来源

## 原则

总准确率只能决定项目还有没有生命，失败模式才决定下一刀往哪里砍。

## 当前样本诊断

- 全量 holdout 准确率：`57.5%`
- 最差切片：
  - `abs_return_5m = mid`：`37.9%`
  - `abs_return_1m = low`：`40.0%`
  - `distance_to_barrier_bps = low`：`40.0%`
- 较强切片：
  - `abs_return_1m = high`：`78.6%`
  - `distance_to_barrier_bps = high`：`78.6%`
  - `model_confidence = 0.80-1.00`：`90.0%`

## 初步解释

模型当前最容易在“短线动量弱、距离 barrier 近、方向不够展开”的场景里失手。下一批候选策略更值得围绕这类场景做门控，而不是继续只调全局置信度阈值。
