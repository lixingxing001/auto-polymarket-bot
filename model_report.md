# Model Report

## 当前结论

- 仅使用分钟级价格行为时，模型存在弱信号，但还没有达到可直接实盘的强度。
- 最近约 24 小时的 287 个已结算样本里，旧基线准确率约 61.3%。
- 最近约 48 小时的 568 个已结算样本里，训练型模型测试集准确率约 63.7%。
- 在 `0.70` 置信度阈值下，测试集精选样本命中率约 70.0%，覆盖率约 40.9%。
- walk-forward 评估中，`0.65` 阈值的平均精选命中率约 72.4%，平均覆盖率约 45.7%。
- 接入 Polymarket 历史价格后，最近 288 个窗口的测试集中，`0.65` 阈值精选命中率约 79.3%，walk-forward 平均精选命中率约 79.4%。

## 对 80% 目标的判断

80% 更像是“少量精选窗口”的候选目标，不适合作为“所有窗口都预测”的硬指标。

如果后续加入逐笔成交、盘口失衡、Polymarket 自身盘口动态后，精选窗口长期样本外仍能稳定接近 80%，那才值得认真讨论扩大仓位。

## 当前最重要的下一步

1. 接入 Polymarket 历史价格，训练 market-aware 模型
2. 持续运行前向纸面交易，积累包含微结构特征的真实样本
3. 将当前 REST 轮询切换到 WebSocket
4. 等前向样本足够后，重新训练并比较：
   - 全量准确率
   - 高置信度命中率
   - 覆盖率
   - 扣费后期望收益
