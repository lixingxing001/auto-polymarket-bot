# Sample Expansion Report

## 新增能力

1. 历史样本缓存
   - 首次抓取后将样本落到本地缓存
   - 同样窗口范围的复跑耗时从分钟级降到约 5 秒

2. 自有盘口快照
   - 每次采集记录：
     - Up / Down best bid
     - Up / Down best ask
     - 对应 size
     - 距离结算秒数
   - 同时生成按窗口聚合的 summary

## 现场验证

同一窗口内的三次采样已经显示：

- Up ask：0.62 到 0.81
- Down ask：0.20 到 0.39

这说明未来回测如果只看事后成交，很容易错过窗口内价格是怎样移动的。

## 当前研究状态

最近 288 个窗口的扩展研究：

- 市场价格基线：57.8%
- 测试集高置信度样本：
  - 命中率：75.9%
  - 覆盖率：33.3%
- walk-forward 高置信度平均命中率：70.3%
